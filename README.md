# SQLGem - SQL ER Diagram Extension

SQLGem is a VS Code extension for creating and managing SQL database ER (Entity-Relationship) diagrams with automatic SQL file generation.

## Features

- **Visual ER Diagrams**: Create and visualize database schemas with an interactive diagram interface
- **Database Management**: Create database folders with organized schema structure
- **Schema Organization**: Add multiple schemas to your database with folder-based organization
- **Table Designer**: Visual table designer with column management
- **MSSQL Support**: Generates SQL scripts using Microsoft SQL Server syntax
- **Dark Mode**: Built-in dark mode support with toggle
- **Drag & Drop**: Move tables around the diagram canvas
- **Auto-Generated SQL**: Automatically creates .sql files for each table in the proper folder structure

## How to Use

### 1. Open the ER Diagram

Run the command **SQLGem: Open ER Diagram** from the Command Palette (Cmd+Shift+P / Ctrl+Shift+P).

### 2. Create a New Database

1. Click the **"New Database Diagram"** button in the toolbar
2. Enter your database name (e.g., "MyDatabase")
3. A folder will be created in your workspace with the database name

### 3. Add Schemas

1. Click the **"Add Schema"** button
2. Enter a schema name (e.g., "dbo", "sales", "hr")
3. A subfolder will be created under your database folder

### 4. Create Tables

1. Click the **"Add Table"** button
2. Select the schema where you want to create the table
3. Enter the table name
4. Add columns with:
   - Column name
   - Data type (INT, VARCHAR, NVARCHAR, TEXT, DATETIME, BIT, DECIMAL)
   - Primary Key checkbox
   - Nullable checkbox
5. Click **"Create Table"**

The table will appear in the diagram and a `.sql` file will be generated in the schema folder.

## Folder Structure

SQLGem creates the following structure in your workspace:

```
MyDatabase/
├── dbo/
│   ├── Users.sql
│   └── Orders.sql
├── sales/
│   └── Products.sql
└── hr/
    └── Employees.sql
```

## Generated SQL Files

Each table generates a MSSQL-compliant SQL file. Example:

```sql
-- Table: dbo.Users
-- Generated by SQLGem

CREATE TABLE [dbo].[Users] (
    [UserId] INT PRIMARY KEY NOT NULL,
    [Username] NVARCHAR(255) NOT NULL,
    [Email] NVARCHAR(255) NOT NULL,
    [CreatedAt] DATETIME NOT NULL
);
```

## Keyboard Shortcuts
